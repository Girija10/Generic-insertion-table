#Steps to run the application
1. Import sample project in eclipse and required actions need to be done(builing the project, etc).
2. Right click project -> Run as Java Application (or) 
mvn spring-boot:run -> run this command in command prompt.
3. Tomcat get started in the specified port.
4. Import MY-APP in visual studio(for front end) and needed action should be done(installing nodejs, etc).
5. Start the server using npm start in terminal.
6. It gets started in the port 4200.
7. Then run localhost:4200/tables in browser.
8. Application shows the table name in the database  and the user can able to click and insert the record in the corresponding table using generic manner.
